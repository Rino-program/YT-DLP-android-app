# yt-dlp ダウンローダー - バッチ処理用URLサンプル
# このファイルのフォーマット:
# - 1行に1つのURL
# - #で始まる行はコメント
# - 空行は無視される
# - YouTubeのプレイリストURL、またはプレイリストに含まれる動画URL

# YouTube動画の例
# https://www.youtube.com/watch?v=dQw4w9WgXcQ
# https://www.youtube.com/watch?v=jNQXAC9IVRw
